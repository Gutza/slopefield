Slopefield Home

A computer art personal project based on slope fields.

This is a hobby project which aims to generate computer art from differential equations approximated numerically based on slope fields. As of March 2016 this is a live, ongoing project: no art proper has been generated yet, but the project is under active development.

The Slopefield project development is a performance art project in itself, in that I'm trying to not only produce tangible code, but I'm also trying to document the process. As such, you can follow the progress in any of the following ways:

  • Facebook: hashtag #slopefield_art
  • Blog: Slopefield blog
  • Source: GitHub repository
  • Working code: GitHub hosted working files (this document), plus some experiments.

The entire project is currently planned as client-only (JavaScript).
